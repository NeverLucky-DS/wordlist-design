# Deutsch Essay · Wörterbuch

Design and prototype for a German essay vocabulary learning app.

## Pages

- `index.html` — word dictionary (Wörterbuch)
- `editor.html` — essay writing editor
- `word-card.html` — word detail card prototype

## Structure

- `css/` — shared and page-specific styles
- `js/` — client-side logic and animations
- `images/` — watercolor and botanical design assets
- `editor-extract/` — extracted editor stack (React frontend + FastAPI backend)
- `PIPELINE.md` — word enrichment pipeline specification
